# Introduction to mobile system Lab 1-3 by Tsimafei Mardashou ID 46011

## Description

This app demonstrates basic functions of a mobile app
- MainActivity
  - Presents text using textView "Hello world" (Lab 1)
  - Greets user in Polish when they put in their name using editText and press a button (Lab 2)
- GuessingGameActivity
  - This is a number guessing game, where user guesses a number 1-10 and tries to get the right answer on the 2nd try exactly.
  - The screen presents:
    - Text views for communication with the user and showing the number of attempts
    - A text input bar for putting in a guess
    - A button to submit an answer
  - The game restarts automatically upon guessing correctly
